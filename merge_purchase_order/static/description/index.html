Merge Purchase Order

The purpose of this module is to Merge the purchase order with are in Request For Quoation State.This module helps us to reduce RFQ for same vendors thus orders can be easily managed

  • If you selected first option 'Create new order and cancel all selected purchase orders'.After click on Merge Orders button, You can see a new order will be created and other selected order will be cancelled.

  • If we go with second type option.After click on Merge Orders button,with second option You can see a new order will be created and other selected order will be deleted.

  • If we goes with third type option.You can see a 'Purchase Orders' field, In this you can select a existing order in which you want to added another selected orders.After click on Merge Orders button, You can see a another orders merge with selected existing order.And rest of all selected orders will be cancelled.

  • If we goes with forth type option.You can see a 'Purchase Orders' field, In this you can select a existing order in which you want to added another selected orders.After click on Merge Orders button, You can see a another orders merge with selected existing order.And rest of all selected orders will be deleted.

Vendors from the Selected PO orders should be same Otherwise this Warning is thrown
