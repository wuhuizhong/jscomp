Product Customer Code

This module will introduce a new feature to manage customer specific product code and product name. those product code and product name will be displayed in description(name) field of product lines.

This description(name) field is display on reports(quotation and invoice).

>> Product with default name and code


>> New field introduced here, 'Customer Code' to maintain list of customer specific product name and product code.


>> When we add order lines in sale-quotation, we can search product with default code/name but we populate product code/name based on selected customer.


>> When we add order lines in sale-quotation, we can search product with default code/name but we populate product code/name based on selected customer. but if no product code/name found for selected customer then it will display default product code/name.


>> When we add sale invoice lines in sale-invoice, we can search product with default code/name but we populate product code/name based on selected customer.


>> After selection of product, product field will have default code/name but description will store customer specific code/name.
